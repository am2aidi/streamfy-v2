# Streamfy 🎵🎬

**Streamfy** is a modern music and video streaming web app built with **Next.js** and **TypeScript**, designed for users to watch movies, anime, music, shorts, and sports, with a personalized experience.

---

## Features

- User authentication and secure login
- Movies, Anime, Music, Shorts, and Sports sections
- Admin panel for managing news, feedback, and community content
- Chatbot for interactive user support
- Personalized watchlist to save favorite content
- Responsive design for desktop and mobile
- Multi-language support (using translations library)
- Social sharing options for content
- Modern UI with clean navigation

---

## Installation

Follow these steps to run Streamfy locally:

```bash
# Clone the repository
git clone https://github.com/am2aidi/streamfy.git

# Navigate into the project folder
cd streamfy

# Install dependencies
npm install

# Start the development server
npm run dev
